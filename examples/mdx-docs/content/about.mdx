# About

This is a **minimal** documentation example demonstrating:

- **MDX format** with frontmatter (title, description, metadata)
- **File-based routing** — each `.mdx` file maps to a route
- **Tailwind-style** utility classes in the layout
- **Pre-rendering** at build time via `build.tish`

## Tech Stack

- Tish (build script + HTTP server)
- Static HTML
- CSS (no JavaScript)
