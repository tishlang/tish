# Tish Docs

Welcome to the **MDX + Tailwind + React** documentation example. This site is:

- Written entirely in **Tish**
- Pre-rendered at build time
- Pure static HTML + CSS (no JavaScript)
- File-based routing

## Quick Start

Run the build, then the server:

```
tish run src/build.tish --features fs,regex
tish run src/main.tish --features http,fs
```

Visit `http://localhost:3000` to see the docs.
